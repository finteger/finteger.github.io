Setting an element to display: flex; establishes a flex container, where its child elements become flex items. By default, flex items will shrink to fit their content width and expand to fill the available space in the container. However, this behavior can be modified using various flex properties such as flex-basis, flex-grow, and flex-shrink.

  • flex-basis sets the initial main size of a flex item. It can be a length (e.g., pixels) or a percentage.
  • flex-grow determines how much a flex item should grow relative to the other flex items in the container when there's extra space available.
  • flex-shrink determines how much a flex item should shrink relative to the other flex items in the container when there's not enough space available.
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Flexbox Example</title>
<style>

    .container {
        display: flex;
        justify-content: center; /* horizontal alignment */
        align-items: center;   /* vertical alignment */
        width: 100%;
        height:200px;
        border: 1px solid #ccc;
    }
    .item {
        flex: 1; /* grow if there's extra space */
        border: 1px solid #333;
    }
    .item1 {
        flex-basis: 100px; /* initial width */
        flex-grow: 1; /* grow if there's extra space */
        flex-shrink: 1; /* shrink if there's not enough space */
    }
    .item2 {
        flex-basis: 200px; /* initial width */
        flex-grow: 2; /* grow twice as much as item1 */
        flex-shrink: 0; /* don't shrink */
    }
     .item3 {
        flex: 0 0 30%;
        /* shorthand: flex: [flex-grow] [flex-shrink] [flex-basis]; */
    }
</style>
</head>
<body>
    <div class="container">
        <div class="item item1">Item 1</div>
        <div class="item item2">Item 2</div>
        <div class="item item3">Item 3</div>
    </div>
</body>
</html>

Setting an element to display: grid; establishes a grid container. The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

  • grid-template-rows: Defines the size of each row in the grid.
  • grid-template-columns: Defines the size of each column in the grid.
  • grid-auto-flow: Determines how auto-placed items are placed in the grid.
  • grid-gap: Sets the size of the gap between grid tracks (rows and columns).
  • align-items: Aligns grid items along the block (column) axis when the items do not use all the space on the cross (row) axis.
  • justify-items: Aligns grid items along the inline (row) axis when the items do not use all the space on the main (column) axis.
  • grid-column: Specifies the starting and/or ending position of a grid item along the inline (row) axis.
  • grid-row: Specifies the starting and/or ending position of a grid item along the block (column) axis.
  • other grid properties: Any other grid-related properties you want to set, such as grid-template-areas, grid-auto-rows, grid-auto-columns, etc.
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>CSS Grid Example</title>
<style>

/* 1fr means one fractional unit.  It's basically the available space which you can add columns or rows.  You can set explicit values like 100px, for instance */


    .container {
        display: grid;
        grid-template-columns: 1fr 1fr; /* Creates two columns with a ratio of 1:1 */
        grid-template-rows: 1fr 1fr 1fr; /* Creates two rows with a ratio of 1:1 */
        grid-gap: 10px; /* Gap between grid items */
    }

    .item1 {
        grid-column: span 1; /* Span across both columns */
        background-color: #ccc;
        padding: 20px;
    }

    .item2 {
        grid-column: 2; /* Start at the second column */
        grid-row: span 3; /* Span across both rows */
        background-color: #ccc;
        padding: 20px;
    }
    
        .item3 {
        grid-column: 1; 
        grid-row: span 2; /* Span across both rows */
        background-color: #ccc;
        padding: 20px;
    }
</style>
</head>
<body>

        <div class="item1">Item 1</div>
        <div class="item2">Item 2</div>
