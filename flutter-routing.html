In this example, we'll create a Flutter app that demonstrates basic routing between two screens using the Navigator widget.

Before starting this example, make sure you have Flutter installed on your system. You can follow the official installation instructions.
  1. Create a new Flutter project: 
    flutter create routing_example
  2. Navigate to the project directory: 
    cd routing_example

First, let's create two screens: HomeScreen and DetailScreen.

HomeScreen

Create a new Dart file named home_screen.dart in the lib directory and add the following code:

import 'package:flutter/material.dart';

class HomeScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Home'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () {
            Navigator.push(
              context,
              MaterialPageRoute(builder: (context) => DetailScreen()),
            );
          },
          child: Text('Go to Detail Screen'),
        ),
      ),
    );
  }
}

Create another Dart file named detail_screen.dart in the lib directory and add the following code:

import 'package:flutter/material.dart';

class DetailScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Detail'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () {
            Navigator.pop(context);
          },
          child: Text('Go back to Home Screen'),
        ),
      ),
    );
  }
}

Open the lib/main.dart file in your Flutter project and replace the existing code with the following:

import 'package:flutter/material.dart';
import 'home_screen.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Routing Example',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: HomeScreen(),
    );
  }
}

In this code:

We import the necessary packages, including material.dart. We define two screens: HomeScreen and DetailScreen. HomeScreen contains a button that navigates to DetailScreen when pressed. DetailScreen contains a button that navigates back to HomeScreen when pressed. In the main.dart file, we set HomeScreen as the initial route of the app.
