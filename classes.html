Classes in Dart are blueprints for creating objects. They can have fields (variables) and methods (functions). You can define a class using the class keyword, and you can create objects from the class using the new keyword or without it in Dart.

  1. Try this out in an editor: 
    void main() {
  // Creating an object from the class
  Person person = Person('Janet', 30);

  // Accessing fields and methods
  print('Name: ${person.name}');
  print('Age: ${person.age}');
  person.sayHello();
}

// Defining a class
class Person {
  // Fields
  String name;
  int age;

  // Constructor
  Person(this.name, this.age);

  // Method
  void sayHello() {
    print('Hello, my name is $name and I am $age years old.');
  }
}

Explanation

  • Fields: Variables inside a class that hold the state of an object.
  • Constructor: A special method used to initialize objects of the class.
  • Methods: Functions defined inside a class that can operate on the object's fields and perform actions.
