Lists in Dart are ordered collections of items. You can create a list using the List class. Dart supports fixed-length and growable lists. Elements in a list can be accessed by their index, starting from 0.

  1. Try this out in an editor: 
    void main() {
  // Creating a list of integers
  List<int> numbers = [1, 2, 3, 4, 5];

  // Adding an element
  numbers.add(6);

  // Accessing elements
  print('First number: ${numbers[0]}');

  // Iterating over the list
  for (int number in numbers) {
    print(number);
  }
}

Explanation

  • List Creation: Lists can be created with initial values or empty. You can also specify the type of elements.
  • Adding Elements: Use the add() method to append elements.
  • Accessing Elements: Access list elements by their index.
  • Iteration: Use a loop to iterate over the list elements.

Maps are collections of key-value pairs. The key is unique, and it is used to access the corresponding value. Dart's Map class supports various methods for adding, accessing, and modifying key-value pairs.

  1. Try this out in an editor: 
    void main() {
  // Creating a map
  Map<String, String> capitals = {
    'USA': 'Washington, D.C.',
    'France': 'Paris',
    'Japan': 'Tokyo',
  };

  // Adding a key-value pair
  capitals['Germany'] = 'Berlin';

  // Accessing a value
  print('Capital of France: ${capitals['France']}');

  // Iterating over the map
  capitals.forEach((key, value) {
    print('The capital of $key is $value.');
  });
}

Explanation

  • Map Creation: Maps are initialized with key-value pairs or empty. Keys must be unique.
  • Adding Key-Value Pairs: Use the assignment operator to add new key-value pairs.
  • Accessing Values: Retrieve values using their corresponding keys.
  • Iteration: Use forEach to iterate over map entries.

Sets are collections of unique items. Dart's Set class supports operations to add, remove, and check for elements. Sets do not allow duplicate values.

  1. Try this out in an editor: 
    void main() {
  // Creating a set of integers
  Set<int> numbers = {1, 2, 3, 4, 5};

  // Adding an element
  numbers.add(6);

  // Removing an element
  numbers.remove(3);

  // Checking if an element is present
  print('Contains 4? ${numbers.contains(4)}');

  // Iterating over the set
  for (int number in numbers) {
    print(number);
  }
}

Explanation

  • Set Creation: Sets are created with initial values or empty.
  • Adding and Removing Elements: Use add() to include an element and remove() to delete one.
  • Checking for Presence: Use contains() to check for the existence of an element.
  • Iteration: Use a loop to iterate over set elements.
