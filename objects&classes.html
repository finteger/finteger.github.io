In Dart, classes are used to create custom types by grouping related properties and methods. Objects are instances of classes, allowing you to create multiple instances with similar characteristics.

  1. Try this out in an editor: 
    void main() {
  // Creating an instance of the Person class
  Person person1 = Person('Alex', 25);
  person1.describe();

  // Creating another instance of the Person class
  Person person2 = Person('Jordan', 30);
  person2.describe();
}

class Person {
  String name;
  int age;

  // Constructor
  Person(this.name, this.age);

  // Method to describe the person
  void describe() {
    print('Name: $name, Age: $age');
  }
}

Explanation

  • Class Definition: The Person class defines properties (name and age) and a method (describe).
  • Constructor: The constructor initializes the properties.
  • Creating Instances: Instances of the class are created using the new keyword (optional in Dart).
  • Method: The describe method prints the properties of the object.
