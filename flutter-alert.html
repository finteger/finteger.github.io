In this codelab, we'll explore how to use the AlertDialog widget in Flutter to display alert messages and gather user input.

Before starting this codelab, make sure you have Flutter installed on your system. You can follow the official installation instructions.

Open the lib/main.dart file in your Flutter project and replace the existing code with the following:

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: AlertDialogCodelab(),
    );
  }
}

class AlertDialogCodelab extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('AlertDialog Codelab'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () {
            showDialog(
              context: context,
              builder: (BuildContext context) {
                return AlertDialog(
                  title: Text('Alert'),
                  content: Text('This is an alert dialog.'),
                  actions: [
                    TextButton(
                      onPressed: () {
                        Navigator.of(context).pop();
                      },
                      child: Text('OK'),
                    ),
                  ],
                );
              },
            );
          },
          child: Text('Show Alert'),
        ),
      ),
    );
  }
}

We define a simple Flutter app with a MyApp widget that extends StatelessWidget. Inside the build method of MyApp, we return a MaterialApp with an AlertDialogCodelab widget as the home. AlertDialogCodelab is a StatelessWidget that displays a Scaffold with an AppBar and a Centered ElevatedButton. When the button is pressed, it shows an AlertDialog with a title, content, and an OK button. The AlertDialog is created using the showDialog function, which takes a context and a builder function. The builder function returns the AlertDialog widget with the desired title, content, and actions. The OK button closes the AlertDialog when pressed.
