Understanding Variables

Variables are used to store data that can be used and manipulated throughout your program.

  1. Create a new Dart file:
    touch variables.dart
  2. Open variables.dart in your favorite editor and add the following code:
    void main() {
  // Declaring variables
  var name = 'John Doe'; // Type inferred as String
  int age = 30; // Explicitly specifying type
  double height = 5.9; // Explicitly specifying type
  bool isStudent = false; // Explicitly specifying type

  // Printing variables
  print('Name: $name');
  print('Age: $age');
  print('Height: $height');
  print('Is Student: $isStudent');
}

Explanation

  • var name = 'John Doe'; declares a variable with the type inferred as String.
  • int age = 30; explicitly declares an int type variable.
  • double height = 5.9; explicitly declares a double type variable.
  • bool isStudent = false; explicitly declares a bool type variable.

Dart supports several built-in data types. Here are the most common ones:

  • String: Used to represent text.
    String greeting = 'Hello, World!';
print(greeting);
  • int: Used to represent integer numbers.
    int year = 2023;
print(year);
  • double: Used to represent floating-point numbers.
    double price = 99.99;
print(price);
  • bool: Used to represent boolean values.
    bool isActive = true;
print(isActive);
  • List: Used to represent a collection of items.
    List<String> fruits = ['Apple', 'Banana', 'Cherry'];
print(fruits);
  • Map: Used to represent a collection of key-value pairs.
    Map<String, int> scores = {
  'Alice': 95,
  'Bob': 85,
  'Charlie': 92
};
print(scores);
  • dynamic: Used when the type is not known at compile time.
    dynamic something = 'Hello';
print(something);
something = 42;
print(something);

Exercise 1: Declare and Print Variables

  1. Declare variables for your favorite book, the number of pages, the price, and whether you've read it or not.
  2. Print each variable.

Example:

void main() {
  String favoriteBook = '1984';
  int pages = 328;
  double price = 15.99;
  bool read = true;

  print('Favorite Book: $favoriteBook');
  print('Pages: $pages');
  print('Price: $price');
  print('Read: $read');
}

First line: "Hello World"
Second line: "I am [age] years old."
Third line: "I am [height] meters tall."
Fourth line: "Is Flutter awesome? [true/false]"
