In this example, we'll create a Flutter app that uses a BottomNavigationBar widget to navigate between multiple screens.

Before starting this example, make sure you have Flutter installed on your system. You can follow the official installation instructions.
  1. Create a new Flutter project: 
    flutter create bottom_navigation_bar_example
  2. Navigate to the project directory: 
    cd bottom_navigation_bar_example

First, let's create three screens: HomeScreen, SearchScreen, and ProfileScreen.

HomeScreen

Create a new Dart file named home_screen.dart in the lib directory and add the following code:

import 'package:flutter/material.dart';

class HomeScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Home'),
      ),
      body: Center(
        child: Text('Home Screen'),
      ),
    );
  }
}

Create another Dart file named search_screen.dart in the lib directory and add the following code:

import 'package:flutter/material.dart';

Open the lib/main.dart file in your Flutter project and replace the existing code with the following:

import 'package:flutter/material.dart';
import 'home_screen.dart';
import 'search_screen.dart';
import 'profile_screen.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Bottom Navigation Bar Example',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: BottomNavigationBarExample(),
    );
  }
}

class BottomNavigationBarExample extends StatefulWidget {
  @override
  _BottomNavigationBarExampleState createState() => _BottomNavigationBarExampleState();
}

class _BottomNavigationBarExampleState extends State<BottomNavigationBarExample> {
  int _selectedIndex = 0;

  final List<Widget> _screens = <Widget>[
    HomeScreen(),
    SearchScreen(),
    ProfileScreen(),
  ];

  void _onItemTapped(int index) {
    setState(() {
      _selectedIndex = index;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: _screens[_selectedIndex],
      bottomNavigationBar: BottomNavigationBar(
        items: const <BottomNavigationBarItem>[
          BottomNavigationBarItem(
            icon: Icon(Icons.home),
            label: 'Home',
          ),
          BottomNavigationBarItem(
            icon: Icon(Icons.search),
            label: 'Search',
          ),
          BottomNavigationBarItem(
            icon: Icon(Icons.person),
            label: 'Profile',
          ),
        ],
        currentIndex: _selectedIndex,
        selectedItemColor: Colors.blue,
        onTap: _onItemTapped,
      ),
    );
  }
}

In this code:

We import the necessary packages, including material.dart. We define three screens: HomeScreen, SearchScreen, and ProfileScreen. BottomNavigationBarExample class is a StatefulWidget that holds the state of the selected index. _selectedIndex variable keeps track of the currently selected screen. _screens list contains the three screens. _onItemTapped method is called when a bottom navigation bar item is tapped, and it updates the selected index. In the build method, we return a Scaffold with a BottomNavigationBar widget and a body that displays the selected screen.
